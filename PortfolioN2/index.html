Career Quest

This project aims to create a fully functional job board platform. The frontend is developed using JavaScript, HTML, and CSS, while the backend is built with Django and the REST Framework. In just two weeks, my partner and I implemented essential features, including personalized accounts with login and sign-in pages. The job application process is automated, utilizing user information for streamlined submissions. We also created a recruiter panel that allows users to create and modify job offers, linking them to both the creator and their company. To enhance security, we fortified the Admin page, providing comprehensive access to all data. Admin users can create, modify, and delete any user, company, or job offer, ensuring robust management of the platform.
